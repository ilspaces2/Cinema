# Cinema

- Приложение для покупки билетов в кинотеатр.
- Используемые технологии:
   * Spring boot 
   * Thymeleaf 
   * Bootstrap
   * JDBC 
   * PostgreSQL and H2
   * Liquibase
   * JUnit
   * Mockito
   * Jcip
